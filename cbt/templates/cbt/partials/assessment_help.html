Assessment creation guideline

Creating assessment requires the following information to be filled in the form

Title

This gives a name or title(as it simply implies) with which it is to be identified. It will allow you distinguish one assessment from the other

Description

Description of assessment explains the reason it is created and might also state the candidates that are to take it.

Institution

If the assessment you are creating is going to be used by the Institution or organisation you are working with, create the organisation and then select from the list of organisations/institutions you have created. Note that you have to opt for institutional assessment type before you get this feature.
